# AI Language Tutor

AI-powered personalized language learning application that helps users improve their English skills through interactive speaking, listening, and reading activities.

## Features

* Speaking practice with AI feedback
* Listening quizzes
* Reading exercises
* AI-based pronunciation evaluation
* Personalized learning experience

## Technologies

* Python
* PySide6 (Desktop UI Framework)
* Grok AI API
* Azure Speech-to-Text
* Supabase (Database & Authentication)
